Debt Helper Final Product
Team 3

What this is
Debt Helper is a simple desktop app prototype that helps a user keep track of multiple debts in one place. The main goal is to make debt tracking less messy than spreadsheets and give a basic payoff strategy view.

Tech used
Python
Tkinter for the desktop UI
SQLite for local data storage
Pandas for CSV importing (if installed)

How to run

Open a terminal in the project folder
Run: python main.py
The app window should open

What works right now
Account creation and login (local)
Add and view debts (basic)
CSV import (if pandas is installed and CSV format is correct)
Basic UI navigation

What is still in progress
Payoff strategy calculations are still being finalized
Payments and reminders are not fully implemented yet
UI polish and extra validation is still being improved

Future improvements
Finish payoff strategy comparison
Add payment tracking and progress reporting
Add reminders for upcoming due dates
More UI polish and better error handling
